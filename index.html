mmmail by lsegal

Mmm, a Minimalist mail library for Ruby. Works with SMTP or sendmail. Provides a one-call method for simple one-off emailing.

Mmmm, a Minimalist mail library for Ruby. Works with SMTP or sendmail. One method call to send out emails. You're done. Easy tastes good. Oh, and it works with Ruby 1.9. Join the discussion: #mmmail on freenode

Dependencies

None.

Install

$ git clone git://github.com/lsegal/mmmail $ cd mmmail $ rake install or use GitHub gems: $ sudo gem install lsegal-mmmail --source http://gems.github.com
